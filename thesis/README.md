# 2D-DTW_for_StructedLight
3次元地形データを構造化光を用いて計測します
# Concert Rover 3D Reconstruction System

## Overview
本プロジェクトは、**スマートフォン（可視光カメラ）だけで3次元位置推定を行うローバーシステム**の開発である。

従来の3D認識（例：顔認証）は赤外線（IR）カメラを用いるが、本研究では以下を実現した：

- 可視光のみで3D推定
- 動き続けるローバー環境での推定
- スマートフォン単体でのリアルタイム処理（60FPS）

---

## Key Contributions（新規性）

### 1. 2次元拡張DTWによる対応点推定
- 通常のDynamic Time Warping（DTW）を**2次元空間に拡張**
- 時系列ではなく、**空間的な対応関係の最適化**に応用
 動きながらでも対応点を追跡可能

---

### 2. 可視光ベースの3次元推定
- IRカメラ不要（Android対応）
- フロントカメラで取得した映像を利用
- Pixel座標 ↔ カメラ座標を統合
- どのスマホでも動く3D測位

---

### 3. 動的環境でのグリッド基準系の構築
- 常に「北方向」を基準としたグリッドを更新
- ローバーが動き続ける前提で設計

 静止前提の従来手法を突破

---

### 4. センサーフュージョンによる走行最適化
以下のデータを統合：

- カメラ映像（前方・フロント）
- モーターDuty比
- 加速度センサ
- エンコーダ
- 重力センサ

Mahalanobis距離を用いた異常検知・左右判定  
最適な回転制御を実現

---

### 5. 軽量画像処理パイプライン
リアルタイム性を重視し、以下を採用：

1. 平滑化（ノイズ除去）
2. 二値化（白黒分離）
3. 骨格化（Skeletonization）
4. 連結性解析

グリッド構造の抽出に成功

※ラプラシアンはノイズ増幅のため不採用

---

### 6. クラウド学習 + エッジ推論
学習パイプライン：

- データ収集（動画・センサー）
- AWS Lambda
- Amazon S3
- Amazon SageMaker

 学習済みモデルをスマホにデプロイ  
 オフラインでも最適化走行可能
